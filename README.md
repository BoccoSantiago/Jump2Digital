## How to install
1. Clone this repository 
    > `git clone https://github.com/BoccoSantiago/Jump2Digital-2022.git`
2. Access the app directory
    > `cd Jump2Digital-2022`
4. Install all 
    >`npm install`
5. Run
    >`npm run dev`
